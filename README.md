Loja Virtual – Front-End 2

Descrição

Projeto desenvolvido na disciplina **Front-End 1**, com o objetivo de aplicar conceitos básicos de **HTML5 e CSS3** na criação de uma página de loja virtual. O sistema simula um ambiente de e-commerce simples, com navegação clara e funcionalidades de cadastro.

Estrutura da Página

Menu de navegação

  * Início
  * Usuários
  * Produtos
  * Página inicial** com mensagem de boas-vindas
  * Categorias de produtos**:

  * Moda
  * Eletrônicos
  * Acessórios
  * Casa & Decoração
  * Seção de promoções com destaque para ofertas

Funcionalidades

* Formulário para **cadastro de clientes**
* Formulário para **cadastro de produtos**
* Organização visual focada na experiência do usuário

 Objetivo do Projeto

Praticar a estruturação de páginas web, criação de formulários, uso de campos de entrada, organização de conteúdo e aplicação de estilos básicos, simulando o funcionamento inicial de uma loja virtual.

Tecnologias Utilizadas

* HTML5
* CSS3
